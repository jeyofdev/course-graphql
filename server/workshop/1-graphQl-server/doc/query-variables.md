### Get single course

```javascript
{
    "courseID" : 1
}
```

### Get courses by title

```javascript
{
  "courseTitle": "JavaScript: Understanding The Weird Parts",
}
```

### Update course topic

```javascript
{
  "id": 1,
  "topic": "JavaScript"
}
```

### Get course with fragments

```javascript
{
    "courseID1": 1,
    "courseID2": 2
}
```

### Add course

```javascript
{
  "title": "boom",
  "author": "jeyofdev",
  "description": "I'm not inspired for the text",
  "topic": "React",
  "url": "https://test.com"
}
```
